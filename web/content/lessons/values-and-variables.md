# 値・変数・演算

プログラムは、値を受け取り、加工し、別の値として出力します。値の種類を区別できると、コードの読み書きが安定します。

## 数値と文字列

```python
print(120 + 30)
print("120" + "30")
```

1行目の `120` と `30` は[数値](term:number)なので、結果は `150` です。2行目は[文字列](term:string)をつないでいるため、結果は `12030` です。

見た目が同じ数字でも、引用符で囲まれているかどうかで意味が変わります。

## 値に名前を付ける

```python
price = 120
count = 3
total = price * count
print(total)
```

[変数](term:variable)は、値に付ける名前です。`price = 120` は、`price` という名前へ `120` を[代入](term:assignment)しています。

変数名は、その値が何を表すのか分かる名前にします。短い問題でも、`a` より `price` の方が意図を追いやすいことがあります。

## 計算する

Pythonでは、次の[算術演算子](term:arithmetic-operator)をよく使います。

| 演算 | 記号 | 例 |
| --- | --- | --- |
| 足し算 | `+` | `3 + 2` |
| 引き算 | `-` | `3 - 2` |
| 掛け算 | `*` | `3 * 2` |
| 割り算 | `/` | `3 / 2` |
| 整数の商 | `//` | `7 // 2` |
| 余り | `%` | `7 % 2` |

複数の計算を組み合わせるときは、途中結果へ名前を付けると確認しやすくなります。

```python
subtotal = 500 * 4
discount = 200
total = subtotal - discount
print(total)
```

計算式を書いたら、簡単な値で手計算し、期待する結果と一致するか確認します。
