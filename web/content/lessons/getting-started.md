# コードを書いて動かす

プログラムは、コンピュータに渡す手順書です。最初は短い手順を書き、実行して、結果を見るところから始めます。

## 最初の1行

```python
print("Hello, Python!")
```

[print](term:print) は、丸かっこの中に書いた値を[標準出力](term:standard-output)へ表示します。`"Hello, Python!"` のように引用符で囲んだ文字は[文字列](term:string)です。

コードを書いたら、まずそのまま実行してください。表示された結果が予想と違った場合は、コードと結果を1文字ずつ見比べます。

## コードにメモを残す

```python
# あいさつを表示する
print("おはよう")
```

`#` から行末までは[コメント](term:comment)です。Pythonはコメントを命令として実行しません。コードの意図を未来の自分や他の人へ伝えるために使います。

## エラーは手がかり

引用符や丸かっこを閉じ忘れると、Pythonは処理を続けられず[エラーメッセージ](term:error-message)を表示します。

```python
print("こんにちは"
```

エラーが出たら、最後の行だけでなく「どのファイルの何行目か」「どんな種類のエラーか」も確認します。エラー文を全部理解する必要はありません。まず行番号を見て、その周辺の記号を確認するだけでも十分です。

## この章の進め方

1. コードを読む
2. 何が表示されるか予想する
3. 実行する
4. 予想と結果を比べる
5. 1か所だけ書き換えて、もう一度実行する

この短い往復が、これから学ぶすべての基礎になります。
